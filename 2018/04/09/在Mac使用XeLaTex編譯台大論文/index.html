在Mac使用XeLaTex編譯台大論文

發表於 2018-04-09 |

Intro.

本篇blog紀錄在mac中，使用XeLaTex編譯台大論文
安裝環境：Sublime 3、MacTex
在最新的Mac版的Sublime 3中，Latextools是已經安裝好的。
相對於之前的版本，安裝過程已經非常簡單，以下做一個紀錄。

安裝MacTex

這一包很大，大概快3GB。

Sublime Text3

在此先預設各位碼農的macbook中都以自帶Sublime ＾_＾
另外也假設各位碼農，Sublime都已經安裝package control，需要靠它來安裝 { LaTeXTools plugin }套件。

安裝完成LaTeXTools plugin以後，環境就大概完成了，
剩下一兩步設定，調整一下即可。

使用Mac自帶的preview作為PDF預覽程式

打開Sublime，右上角
Sublime -> Preferences -> Package Settings -> LaTeXTools -> Settings-User
會開啟 LaTexTools.sublime-settings，

若不想用Preview預覽可選擇
Install skim (不想用preview的人選用)

下載台大模板

修改 thesis.tex 文件

在第一行 加上%!TEX program = xelatex，
這樣才會使用xelatex編譯

將\setCJKmainfont{標楷體}
改成 \setCJKmainfont{BiauKai}
然後按cmd +B 編譯thesis.tex 這個文件，就會生成論文了。

更多論文撰寫細節請參考，黃子桓(https://github.com/tzhuan/ntu-thesis/wiki)的Github wiki說明撰寫你的論文

「TIPS」先存檔在編譯，壹定要先存檔

（補充理解用）：
Subime3 已經自帶xelatex
解決漢化問題很簡單：先在文件開始加%!TEX program = xelatex
同時調用package就可以了： \usepackage[UTF8]{ctex}
